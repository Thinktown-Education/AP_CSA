8) Programming: Determine whether a string is a palindrome

Description: Write a method or small program that reads a single string and determines whether it is a palindrome. A palindrome reads the same forwards and backwards. For this exercise ignore letter case and ignore non-letter characters (spaces, punctuation). For example, "Race car!" should be considered a palindrome because when you remove non-letters and compare case-insensitively it reads "racecar" both ways.

Requirements:
- Implement a method `boolean isPalindrome(String s)` that returns `true` when `s` is a palindrome by the rules above, otherwise `false`.
- Provide a short `main` that tests several example inputs and prints results.

Sample inputs and expected output (one per line):

```text
Input: "racecar" -> Palindrome: true
Input: "hello"   -> Palindrome: false
Input: "Race car!" -> Palindrome: true
Input: "A man, a plan, a canal: Panama" -> Palindrome: true
Input: "" -> Palindrome: true
```
