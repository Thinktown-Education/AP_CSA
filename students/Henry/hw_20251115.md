8) Programming: Determine whether a string is a palindrome

Description: Write a method or small program that reads a single string and determines whether it is a palindrome. A palindrome reads the same forwards and backwards. For this version assume all input strings contain only lowercase alphabetic characters (`a`-`z`) with no spaces, punctuation, or uppercase letters â€” students do not need to handle case or special characters.

Requirements:
- Implement `boolean isPalindrome(String s)` that returns `true` when `s` is a palindrome under the simplified assumption.
- Provide a short `main` that tests several example inputs and prints results.

Sample inputs and expected output (one per line):

```text
Input: "racecar" -> Palindrome: true
Input: "hello"   -> Palindrome: false
Input: "level"   -> Palindrome: true
Input: "abca"    -> Palindrome: false
Input: ""        -> Palindrome: true
```

Hint: With the simplified input assumption you can directly compare characters from both ends using two indices moving inward.
